Efectos de wired

Efectos wired

¿Cómo funciona el efecto?

EFECTO WIRED: Cambiar el estado del furni

Este efecto cambia el estado del furni como si estuviesen dándole clic dos veces.

EFECTO WIRED: Teletransportarse al furni

El efecto teletransporta a un usuario hacia un furni, si seleccionar más de un furni el usuario es teletransportado aleatoriamente a uno de ellos.

EFECTO WIRED: Bot que se teletransporta al furni

Este efecto teletransporta a un bot al furni seleccionado anteriormente

EFECTO WIRED: Bot que cambia de ropa

Este efecto le cambio la ropa al bot

EFECTO WIRED: Bot que va al furni

Este efecto hace que el bot se mueva hacia uno de los furnis seleccionados

EFECTO WIRED: Bot que parloteo o susurra a usuarios

Este efecto hace que el bot hable o susurre a un usuario

EFECTO WIRED: Bot que parlotea o susurra

Este efecto hace que el bot hable o susurre al usuario seleccionado

EFECTO WIRED: El bot te sigue

Este efecto hace que el bot siga a un usuario seleccionado

EFECTO WIRED: Bot da item de mano

Este efecto hace que el bot le otorgue un item a un usuario

EFECTO WIRED: echar a un usuario

Este efecto hace que un usuario sea echado de la sala

EFECTO WIRED: Mostrar mensaje

Este efecto mostrará un mensaje secreto para cada usuario que entre a la sala

EFECTO WIRED: Mover y rotar el furni

Este efecto hace que un furni se mueva de lugar o incluse rote de dirección

EFECTO WIRED: Furnis que vuelven a su programación original

Este efecto hace que un furni seleccionado vuelva a su modo original

EFECTO WIRED: Montonar el wired

Este wired le permite seleccionar 5 pilas de wired para ejecutar desde el mismo disparador, independientemente de las condiciones o disparadores actuales.

EFECTO WIRED:: Reiniciar tiempo

Este efecto reinicia los activadores que usan temporizadores.

EFEITO WIRED: Dar Puntos

Este efecto da X cantidad de puntos a los jugadores. ¡Puedes crear grandes juegos!

EXTRA WIRED: Efecto aleatorio

Solo un efecto aleatorio se podrá hacer

EXTRA WIRED: Efecto inédito

Solo un efecto inédito se podrá hacer

EFECTO WIRED: Perseguir

Este efecto mueve el furni al usuario más cercano en la misma fila o columna. Si tienes un usuario cercano a este furni WIRED: se activará

EFECTO WIRED: Irse

Cuando el furni despedido va más allá del avatar más cercano

EFECTO WIRED: Puntuación recibida

Da una puntuación predefinida para el equipo

EFECTO WIRED: Entrar al equipo

El usuario entrará a un equipo predefinido

EFECTO WIRED: Salir del equipo

El usuario abandonará el equipo predefinido

EFECTO WIRED: Cambiar la dirección del furni

Este efecto cambia la dirección del furni. Cuando se bloquea el movimiento, el furni cambiará su dirección de acuerdo con su configuración. Si hay un usuario cerca del furni, se activará Wired Activator
