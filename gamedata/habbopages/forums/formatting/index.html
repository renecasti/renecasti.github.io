h1. Formato de los foros

Formato básico

Use *bold* para que el texto esté en negrito, _italic_ - para texto itálico.

Comience una línea con el símbolo > para hacer una citación, como:
> esta es una citación.

Referente

Para citar a otro usuario simplemente use el formato ‘@Quabbosname’. Eso se transforma en un vínculo direccionado al perfil del usuario y el usuario recibirá la notificación. (Todavía está en fase beta).

Respuesta

Usted podrá responder los mensajes individuales dándole clic al botón verde "Responder" de la barra de mensajes. También puede responde varios mensajes al mismo tiempo, dejando la ventana "Escribir mensaje" abierta, clique en más botones "Responder" en la página principal del foro
