Activadores wired

¿Cómo funciona el activador?

ACTIVADOR WIRED: Usuario que anda en el furni

Este efecto es activado cuando un usuario se sienta o pisa un furni

ACTIVADOR WIRED: Usuario que sale del furni

Este efecto es activado cuando un usuario sale del furni

ACTIVADOR WIRED: Bot ve hacia un usuario

Este efecto es activado cuando algún bot alcance a un usuario

ACTIVADOR WIRED: Bot ve hacia un furni

Este efecto es activado cuando algún bot alcanza a un furni seleccionado

ACTIVADOR WIRED: Usuario parlotea

Este efecto es activado cuando un usuario dice una palabra o frase

ACTIVADOR WIRED: Estado del furni es alterado

Este efecto es activado cuando un furni es alterado dándole clic dos veces

ACTIVADOR WIRED: Usuario entra a la sala

Este efecto es activado cuando un usuario entra a la sala

ACTIVADOR WIRED: Repetición del efecto

Este efecto es activado cada X segundos

ACTIVADOR WIRED: En el tiempo escogido

El efecto se activa a una hora establecida de reiniciar el contador

ACTIVADOR WIRED: Puntuación obtenida

Este efecto se activa cuando se logra una puntuación establecida

ACTIVADOR WIRED: Fin del juego

El efecto es activado cuando el juego termina

ACTIVADOR WIRED: Comienza el juego

El efecto es activado cuando el juego comienza

ACTIVADOR WIRED: Golpear a un usuario

Este efecto se activa cuando un furni (usando wired perseguir) golpea a un usuario

ACTIVADOR WIRED: Periodo de duración

¡Juegos más largos! Los afectados se disparan después de cierto tiempo. ¡Disparando periódicamente, pero ya no!
